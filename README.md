# Enterprise Network Infrastructure Lab

Hands-on Cisco Packet Tracer labs documenting my journey in networking, infrastructure support, and CCNA fundamentals.

---

## Project Overview

The **Enterprise Network Infrastructure Lab** is a practical networking project built using **Cisco Packet Tracer** and inspired by the **Jeremy's IT Lab CCNA course**.

This repository documents my progression from networking fundamentals to enterprise-level concepts through designing, configuring, troubleshooting, and securing network infrastructures in a simulated environment.

The final project simulates the network architecture of **LionTech Solutions Pte Ltd**, a fictional Singapore-based company with multiple departments and enterprise networking requirements.

---

# Project Objectives

This project aims to:

- Strengthen networking fundamentals and CCNA concepts.
- Gain hands-on experience with Cisco devices and protocols.
- Develop structured troubleshooting and problem-solving skills.
- Understand how enterprise networks are designed and secured.
- Build practical experience through simulation and documentation.

# Troubleshooting Case Studies

In addition to network deployment, this repository contains troubleshooting scenarios based on common infrastructure failures.

Each case study follows a structured methodology:

1. Problem Identification
2. Root Cause Analysis
3. Diagnostic Process
4. Solution Implementation
5. Verification
6. Lessons Learned

The objective is to develop analytical thinking and reinforce practical troubleshooting skills commonly used by network administrators and IT support professionals.
